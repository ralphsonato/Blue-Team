# Estruturando um Projeto de SOC/SIEM

Minhas anotações e estudos sobre como estruturar um projeto de SOC e SIEM do zero - desde o levantamento de requisitos até a estruturação final do projeto.

---

## Etapas Macro do Projeto

O projeto segue um ciclo bem definido. Pular qualquer etapa tende a gerar retrabalho caro lá na frente.

```
Levantamento de Requisitos
         ↓
  Definir Arquitetura
         ↓
     Executar RFI
         ↓
  Estruturar a RFP
         ↓
   Executar a PoC
         ↓
 Analisar Resultados da PoC
         ↓
  Avaliar Fornecedores
         ↓
Matriz de Comparativo Técnico
         ↓
   Escolher Fornecedor
         ↓
  Estruturar o Projeto
```
